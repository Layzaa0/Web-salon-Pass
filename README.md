Briefing – Desenvolvimento do Site WebPass
1. Informações Gerais

Nome do projeto: Web salon Pass
Tipo de site: Sistema web simples de agendamento
Segmento: Salão de beleza
Tecnologias: HTML, CSS, JavaScript, LocalStorage
Escopo: Front-end

2. Objetivo do Site

Permitir que clientes realizem um agendamento simples, informando dados básicos como nome, serviço e horário.

3. Público-Alvo

Clientes de salão de beleza

Pessoas que buscam agendamento rápido e simples

Público geral, sem necessidade de conhecimento técnico

4. Funcionalidades Principais

Página inicial com apresentação do sistema

Página de agendamento para:

Nome do cliente

Data selecionada

Serviço selecionado

Horário disponível

Salvamento dos dados no LocalStorage

Navegação entre páginas

5. Estrutura do Site

index.html – Página inicial

agendamento.html – Formulário de agendamento

style.css – Estilos do site

script.js – Lógica e LocalStorage

6. Conteúdo Dinâmico

Serviço selecionado é passado entre páginas via LocalStorage

Dados do agendamento são armazenados no navegador

7. Design (Aparência)

Layout simples e limpo

Cores suaves

Foco na usabilidade

Interface intuitiva, sem excesso de elementos

8. Critério de Aceite do Projeto

O projeto será considerado concluído quando:

O usuário conseguir navegar entre as páginas

Selecionar um serviço

Informar os dados do agendamento

Os dados forem salvos corretamente no LocalStorage

9. Observações

Projeto com finalidade educacional, voltado para aprendizado de desenvolvimento web básico e lógica de front-end.
